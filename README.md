BassProject
===========

A starting point for new web projects using [Basscss](http://www.basscss.com/). [Docs](http://basscss.com/docs)

* grunt
* stylesheets
	* mixins.scss
	* basscss.scss - broken out into individual modules
* javascript
	* modernizr custom build
